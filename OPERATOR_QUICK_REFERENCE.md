# Operator Quick Reference Card

## 🎯 What You Can Do

### On Every Job Card:

1. **Update Status** ⚡
   - Change job status (Assigned → Traveling → Operating)
   - Add notes about progress
   - Notifies admins automatically

2. **Submit Harvest** 🌾 (IN-KIND only)
   - Enter total harvest in sacks
   - System calculates BUFIA share (÷ 9)
   - Notifies admins for rice delivery

## 📍 Navigation Links

| Link | What It Shows | When to Use |
|------|---------------|-------------|
| **Dashboard** | All your jobs | See everything at once |
| **All Assigned Jobs** | All your jobs | Same as Dashboard |
| **In Progress** | Jobs you're working on now | Focus on active work |
| **Awaiting Harvest** | IN-KIND jobs needing reports | Know what needs harvest reports |
| **View Machines** | All equipment | Check machine details |

## 🔢 Job Statuses

1. **Assigned** - Job given to you, not started
2. **Traveling** - On the way to location
3. **Operating** - Currently working
4. **Harvest Reported** - Harvest submitted (auto-set)

## 📊 Statistics Cards

Always show counts for ALL your jobs:
- **Assigned**: Total jobs assigned to you
- **In Progress**: Jobs you're traveling to or operating
- **Awaiting Harvest**: IN-KIND jobs ready for harvest report
- **Completed**: Jobs with submitted harvest reports

## 🌾 Harvest Calculation

```
Total Harvest: 21.3 sacks
BUFIA Share: 2.37 sacks (21.3 ÷ 9)
Member Share: 18.93 sacks (21.3 - 2.37)
```

System calculates automatically - you just enter the total!

## ⚠️ Important Notes

- You can ONLY see YOUR assigned jobs
- Harvest reports are ONLY for IN-KIND rentals
- Status updates notify admins automatically
- Navigation bar stays visible on all pages

## 🆘 Common Issues

**Can't submit harvest?**
- Check if payment type is IN-KIND
- Make sure you haven't already submitted
- Verify you're in "Operating" status

**No jobs showing?**
- You haven't been assigned any jobs yet
- Contact admin to assign jobs

**Navigation disappeared?**
- Fixed in latest update
- Clear browser cache if issue persists
