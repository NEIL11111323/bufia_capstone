How to Create Operator Accounts

Use one of these setup methods to add new operators to the system.

Method 1: Script

python create_operator.py

Recommended for quick local setup.

Method 2: Django Admin

Go to `/admin/`, add a user, then set the role to `Operator` and enable staff status.

Method 3: Django Shell

python manage.py shell

Create the operator account programmatically when needed.
