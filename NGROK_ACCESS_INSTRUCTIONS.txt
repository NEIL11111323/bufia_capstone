BUFIA NGROK SETUP
=================

The screen in your screenshot is the normal ngrok help screen.
It appears when `ngrok.exe` is opened without a command.

To expose this Django app, use ngrok with the local BUFIA port:

1. Start Django:
   `.\run_local.ps1`

2. In a second terminal, start ngrok:
   `.\start_ngrok.ps1`

3. If ngrok asks for authentication first, run this once:
   `C:\Users\Admin\Downloads\ngrok-v3-stable-windows-amd64\ngrok.exe config add-authtoken YOUR_TOKEN`

4. After ngrok starts, copy the HTTPS forwarding URL and open it in a browser.

Notes
-----
- BUFIA runs on `http://127.0.0.1:8000`
- This project already allows `*.ngrok-free.app` and `*.ngrok.io`
- Do not run plain `ngrok.exe` by itself; use `ngrok.exe http 8000`
