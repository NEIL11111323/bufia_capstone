# 🔄 HOW TO CLEAR BROWSER CACHE - STEP BY STEP

## Why You Need to Do This

Your browser has saved (cached) the old navigation from the previous version. Even though we've fixed all the templates, your browser is still showing you the old cached version. You MUST clear the cache to see the new clean navigation.

## Quick Fix: Hard Refresh (Try This First)

### Windows Users:
1. Go to the operator dashboard page
2. Press **`Ctrl + Shift + R`** (hold all 3 keys together)
   - OR press **`Ctrl + F5`**
3. Wait for page to reload
4. Check if you see the new clean navigation

### Mac Users:
1. Go to the operator dashboard page
2. Press **`Cmd + Shift + R`** (hold all 3 keys together)
3. Wait for page to reload
4. Check if you see the new clean navigation

---

## Full Cache Clear (If Hard Refresh Doesn't Work)

### Google Chrome / Microsoft Edge

1. **Open Settings**
   - Click the 3 dots (⋮) in top-right corner
   - Click "Settings"

2. **Go to Privacy and Security**
   - In the left sidebar, click "Privacy and security"
   - Click "Clear browsing data"

3. **Select What to Clear**
   - Choose "Last hour" or "All time"
   - ✅ Check "Cached images and files"
   - ✅ Check "Cookies and other site data" (optional but recommended)
   - ❌ Uncheck "Browsing history" (unless you want to clear it)

4. **Clear Data**
   - Click "Clear data" button
   - Wait for confirmation

5. **Close and Reopen Browser**
   - Close ALL browser windows
   - Reopen browser
   - Go to BUFIA website
   - Log in again

### Mozilla Firefox

1. **Open Settings**
   - Click the 3 lines (≡) in top-right corner
   - Click "Settings"

2. **Go to Privacy & Security**
   - In the left sidebar, click "Privacy & Security"
   - Scroll down to "Cookies and Site Data"

3. **Clear Data**
   - Click "Clear Data..." button
   - ✅ Check "Cached Web Content"
   - ✅ Check "Cookies and Site Data" (optional)
   - Click "Clear"

4. **Close and Reopen Browser**
   - Close ALL browser windows
   - Reopen browser
   - Go to BUFIA website
   - Log in again

### Safari (Mac)

1. **Open Preferences**
   - Click "Safari" in menu bar
   - Click "Preferences"

2. **Go to Advanced**
   - Click "Advanced" tab
   - ✅ Check "Show Develop menu in menu bar"

3. **Clear Cache**
   - Click "Develop" in menu bar
   - Click "Empty Caches"

4. **Close and Reopen Browser**
   - Close ALL browser windows
   - Reopen browser
   - Go to BUFIA website
   - Log in again

## Still Not Working?

### Try These Steps:

1. **Log Out Completely**
   - Click logout
   - Close ALL browser tabs
   - Close browser completely
   - Wait 10 seconds

2. **Reopen and Log In**
   - Open browser fresh
   - Go to BUFIA website
   - Log in as operator

3. **Try Different Browser**
   - If using Chrome, try Firefox
   - If using Firefox, try Chrome
   - Fresh browser = no cache

4. **Check Browser Console**
   - Press **F12** on keyboard
   - Click "Console" tab
   - Take screenshot of any red errors
   - Send to support

5. **Contact Support**
   - Provide your username
   - Provide browser name and version
   - Provide screenshot of what you see
   - Mention you've cleared cache

## Why This Happens

Browsers save (cache) website files to load pages faster. When we update the website, your browser doesn't know about the changes and keeps showing the old saved version. Clearing the cache forces the browser to download the new files.

Think of it like this:
- 📚 Your browser has an old textbook (cached files)
- 📖 We've published a new edition (updated templates)
- 🔄 You need to get the new book (clear cache)
- ✅ Then you'll see the updated content
