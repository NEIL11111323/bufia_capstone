# 🚀 Operator Quick Start Guide

## Login

1. Go to: `http://your-domain.com/accounts/login/`
2. Enter credentials:
   - Username: `operator`
   - Password: `operator123`
3. Click "Login"

## Navigation

After login, you'll see the operator menu in the sidebar:

### Main Pages

1. **Dashboard** (`/machines/operator/dashboard/`)
   - View statistics (Active, In Progress, Completed jobs)
   - See recent jobs
   - Quick overview

2. **All Jobs** (`/machines/operator/jobs/`)
   - List of all assigned jobs
   - View job details
   - Filter by status

3. **Ongoing Jobs** (`/machines/operator/jobs/ongoing/`)
   - Jobs currently in progress
   - Update job status
   - Add notes

4. **Awaiting Harvest** (`/machines/operator/jobs/awaiting-harvest/`)
   - In-kind payment jobs ready for harvest
   - Submit harvest reports
   - Track harvest status

5. **Completed Jobs** (`/machines/operator/jobs/completed/`)
   - History of finished jobs
   - View results

6. **View Machines** (`/machines/operator/machines/`)
   - See all equipment
   - Check machine status
   - View specifications

7. **Notifications** (`/machines/operator/notifications/`)
   - Job assignments
   - Status updates
   - Admin messages

## Common Tasks

### Update Job Status

1. Go to "All Jobs" or "Ongoing Jobs"
2. Click "View Details" on a job
3. Select new status:
   - **Assigned** - Job assigned, not started
   - **Traveling** - On the way to site
   - **Operating** - Currently working
   - **Harvest Ready** - Ready for harvest (in-kind only)
4. Add notes (optional)
5. Click "Update Status"

### Submit Harvest Report

For in-kind payment jobs:

1. Go to "Awaiting Harvest"
2. Find the job
3. Enter total harvest in sacks
4. Add notes (optional)
5. Click "Submit Report"
6. Admin will review and approve

### Check Notifications

1. Click bell icon in top navbar (shows unread count)
2. Or go to "Notifications" in sidebar
3. Filter by type:
   - All
   - Unread
   - Job Assignments
   - Harvest
   - Urgent
4. Click notification to mark as read

### View Job Details

1. Go to any job list page
2. Click "View Details" button
3. See complete job information:
   - Machine details
   - Member information
   - Location and area
   - Payment type
   - Current status
4. Update status or submit harvest from detail page

## Job Status Flow

```
Assigned → Traveling → Operating → Harvest Ready → Completed
```

- **Assigned**: Job assigned to you
- **Traveling**: On the way to job site
- **Operating**: Currently working on the job
- **Harvest Ready**: Job done, ready for harvest (in-kind only)
- **Completed**: Job finished (set by admin)

## Payment Types

### Cash Payment
- Member pays upfront
- Complete the job
- Update status to "Operating"
- Admin marks as completed

### In-Kind Payment
- Member pays with harvest share
- Complete the job
- Update status to "Harvest Ready"
- Submit harvest report
- Admin reviews and approves
- Admin marks as completed

## Tips

1. **Update status regularly** - Keep admin informed
2. **Add notes** - Explain any issues or delays
3. **Check notifications** - Stay updated on assignments
4. **Submit harvest reports promptly** - Don't delay reporting
5. **Use mobile** - Interface is mobile-friendly for field use

## Troubleshooting

### Can't see operator menu?
- Make sure you're logged in as operator
- Check your role is set to "operator"
- Clear browser cache (Ctrl+Shift+Delete)

### No jobs showing?
- Admin needs to assign jobs to you
- Check "All Jobs" to see if any exist
- Contact admin if you should have jobs

### Can't update status?
- Make sure job is not completed/cancelled
- Check you're assigned to this job
- Try refreshing the page

### Harvest form not showing?
- Only for in-kind payment jobs
- Job must be in "Operating" or "Harvest Ready" status
- Check payment type is "In-Kind"
