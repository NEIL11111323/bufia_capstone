### New Member Registration and Verification
1. User creates an account in the system
2. User fills out the detailed membership application form
3. User submits the application and waits for verification
4. Administrators review the application
5. Administrator approves or rejects the application
6. If approved, user gains full access to system features
7. If rejected, user can update information and resubmit

## Membership Policy Rules
1. All new users must complete membership verification to access full system features
2. Only verified members can rent machines or make scheduling requests
3. Unverified members can only:
   - Edit their profile information
   - Submit/resubmit membership applications
   - View public information
4. Verification requires submission of complete and accurate membership information
5. Administrators must approve membership applications before verification is complete
6. Rejected applications require administrator feedback
7. Presidents and superusers are exempt from verification requirements
